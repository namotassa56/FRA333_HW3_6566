### ‡∏™‡∏°‡∏Å‡∏≤‡∏£‡∏ó‡∏µ‡πà‡πÄ‡∏Å‡∏µ‡πà‡∏¢‡∏ß‡∏Ç‡πâ‡∏≠‡∏á

### Jacobian Matrix $$ \mathbf{J} $$
Jacobian Matrix ‡πÅ‡∏™‡∏î‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡∏™‡∏±‡∏°‡∏û‡∏±‡∏ô‡∏ò‡πå‡∏£‡∏∞‡∏´‡∏ß‡πà‡∏≤‡∏á **‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏°‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠** ‡∏Å‡∏±‡∏ö **‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô‡πÅ‡∏•‡∏∞‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏°‡∏Ç‡∏≠‡∏á‡∏õ‡∏•‡∏≤‡∏¢‡∏°‡∏∑‡∏≠‡∏´‡∏∏‡πà‡∏ô‡∏¢‡∏ô‡∏ï‡πå (End-effector)**.

$$
\mathbf{J} = 
\begin{bmatrix}
\mathbf{J}_v \\ 
\mathbf{J}_\omega
\end{bmatrix}
$$


- $$ \mathbf{J}_v $$: ‡πÉ‡∏ä‡πâ‡πÅ‡∏õ‡∏•‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏°‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡πÉ‡∏´‡πâ‡πÄ‡∏õ‡πá‡∏ô **‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô** ‡∏Ç‡∏≠‡∏á‡∏õ‡∏•‡∏≤‡∏¢‡∏°‡∏∑‡∏≠‡∏´‡∏∏‡πà‡∏ô‡∏¢‡∏ô‡∏ï‡πå
- $$ \mathbf{J}_\omega $$: ‡πÉ‡∏ä‡πâ‡πÅ‡∏õ‡∏•‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏°‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡πÉ‡∏´‡πâ‡πÄ‡∏õ‡πá‡∏ô **‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏°** ‡∏Ç‡∏≠‡∏á‡∏õ‡∏•‡∏≤‡∏¢‡∏°‡∏∑‡∏≠‡∏´‡∏∏‡πà‡∏ô‡∏¢‡∏ô‡∏ï‡πå

### ‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô

$$
\mathbf{v}_i = \mathbf{z}_i \times (\mathbf{p}_e - \mathbf{p}_i)
\]

- \( \mathbf{z}_i \): ‡πÅ‡∏Å‡∏ô‡∏´‡∏°‡∏∏‡∏ô‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠ \(i\) (‡πÅ‡∏Å‡∏ô \(z\) ‡∏Ç‡∏≠‡∏á‡πÄ‡∏ü‡∏£‡∏°‡∏ó‡πâ‡∏≠‡∏á‡∏ñ‡∏¥‡πà‡∏ô)  
- \( \mathbf{p}_e \): ‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡∏Ç‡∏≠‡∏á‡∏õ‡∏•‡∏≤‡∏¢‡∏°‡∏∑‡∏≠‡∏´‡∏∏‡πà‡∏ô‡∏¢‡∏ô‡∏ï‡πå  
- \( \mathbf{p}_i \): ‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠ \(i\)  
- \( \times \): ‡∏Å‡∏≤‡∏£‡∏Ñ‡∏π‡∏ì‡πÅ‡∏ö‡∏ö **Cross Product**

$$

### ‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏°

$$

- ‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏°‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠ \(i\) ‡πÄ‡∏ó‡πà‡∏≤‡∏Å‡∏±‡∏ö‡∏ó‡∏¥‡∏®‡∏ó‡∏≤‡∏á‡∏Ç‡∏≠‡∏á‡πÅ‡∏Å‡∏ô \(z\) ‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏ô‡∏±‡πâ‡∏ô
$$

‡∏Å‡∏≤‡∏£‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì Jacobian ‡πÉ‡∏ô Python

```bash

    J = np.zeros((6, n))
    
    # ‡∏Å‡∏≥‡∏´‡∏ô‡∏î‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡∏Ç‡∏≠‡∏á‡πÄ‡∏ü‡∏£‡∏°‡∏ó‡∏µ‡πà 0 (‡∏ê‡∏≤‡∏ô)

    # ‡∏ß‡∏ô‡∏•‡∏π‡∏õ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡πÅ‡∏ï‡πà‡∏•‡∏∞‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠
    for i in range(n):
        # ‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏ó‡∏µ‡πà i

        else:
            p_i = P[:, i]  # ‡πÉ‡∏ä‡πâ‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏ó‡∏µ‡πà i-1
        
        # ‡πÅ‡∏Å‡∏ô z ‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏ó‡∏µ‡πà i (‡πÅ‡∏Å‡∏ô‡∏Å‡∏≤‡∏£‡∏´‡∏°‡∏∏‡∏ô)
        z_i = R[:, 2, i]  # ‡πÅ‡∏Å‡∏ô z ‡∏Ç‡∏≠‡∏á‡πÄ‡∏ü‡∏£‡∏°‡∏ó‡∏µ‡πà i (‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå‡∏ó‡∏µ‡πà‡∏™‡∏≤‡∏°‡∏Ç‡∏≠‡∏á‡πÄ‡∏°‡∏ó‡∏£‡∏¥‡∏Å‡∏ã‡πå‡∏Å‡∏≤‡∏£‡∏´‡∏°‡∏∏‡∏ô)
        
        # ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô (Translational Jacobian): Jv = z_i √ó (p_e - p_i)
        J[:3, i] = np.cross(z_i, (p_e - p_i))  # Cross product
        
        # ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏° (Rotational Jacobian): Jw = z_i (‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏´‡∏°‡∏∏‡∏ô)
        J[3:, i] = z_i  # ‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏°
    
     
    # ‡πÅ‡∏õ‡∏•‡∏á‡∏™‡πà‡∏ß‡∏ô‡∏Ç‡∏≠‡∏á rotational Jacobian ‡πÄ‡∏ó‡πà‡∏≤‡∏ô‡∏±‡πâ‡∏ô ‡πÇ‡∏î‡∏¢‡πÉ‡∏ä‡πâ‡∏Å‡∏≤‡∏£ dot product ‡∏Å‡∏±‡∏ö R_e
    J_e_translational = np.dot(R_e.T, J[:3, :])  # Transform the translational part
    J_e_rotational = np.dot(R_e.T, J[3:, :])    # Transform the rotational part

    J_e = np.vstack((J_e_translational, J_e_rotational))

    return J_e
    ```

    print(f"Determinant ‡∏Ç‡∏≠‡∏á Reduced Jacobian: {det_J}")

    else:
        return 0  # ‡πÑ‡∏°‡πà‡∏≠‡∏¢‡∏π‡πà‡πÉ‡∏ô‡∏™‡∏ñ‡∏≤‡∏ô‡∏∞ Singularity
